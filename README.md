# VanTrans82

A Nuxt 3 project with TailwindCSS.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
